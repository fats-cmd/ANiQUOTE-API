Introduction

Welcome to the Anime Quotes API. This API allows you to fetch quotes from various anime, search for quotes by keywords, retrieve random quotes, and more.

Base URL:

https://aniquoteapi/api/v1.com

Endpoints

Fetch quotes by Anime name

GET /api/v1/quotes/anime

Parameters

anime_name string The name of the anime eg: One piece

Sample request:

GET /api/v1/quotes/anime?name=One piece

Get Quote by ID

GET /api/v1/quotes/ID

Parameters

id integer The unique identifier of the quote

Sample request:

GET /api/v1/quotes/ID?id=123

Fetch Random Quotes

GET /api/v1/quotes/random

Fetch quotes by Character name

GET /api/v1/quotes/character

Parameters

ch_name string The name of the character (e.g., "Monkey D. Luffy", "Goku")

Sample request:

GET /api/v1/quotes/character?ch_name=levi

Search Quotes by Keyword

GET /api/v1/quotes/search

Parameters

text string Search text matching quotes, anime names, or character names

Sample request:

GET /api/v1/quotes/search?text=heavy
